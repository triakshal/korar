Simple CLI to-do list manager in python. 


Usage: 

python korar.py [command] [options]

commands: 

list: 
list

add: 
add [description]

remove:
remove [number]

update:
update [number] [new description]

mark-complete:
mark-complete [number]

mark-in-progress:
mark-in-progress [number]
